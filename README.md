### What I learned

I learned how to place an image in a specific location using absolute and relative positioning.

```css
    .quotation{
        display: block;
        width: 104px;
        height: 102px;
        position: absolute;
        top: 0px;
        right: 190px;
        z-index: 10;
    }
    .testimonies{
        position: relative;
        z-index: 100;
    }

```
